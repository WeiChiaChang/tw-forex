> Search foreign currency via interactive cli

## Install
```shell
$ npm i tw-forex -g
```

## Usage
```shell
Search foreign currency via interactive cli
  $ tw-forex

  Search specified currency
  $ tw-forex -s

  Show current version
  $ tw-forex -v

  Source code of this side project
  $ tw-forex -o
```
